# Description

Vietnam defender is a classic game of kill enemies in waves, by killing enemies the game becomes more dificult, but donÂ´t worry about it you have 3 extra lives and 3 mines to spare.

# Main Functionalities

- 4D Movement
- Colision with moving objects
- FPS control
- Music & effects control
- Fire button
- Extra weapon
- Lives system
- HighScore system
